# Smart-PYQs
Smart PYQ Quiz is an intelligent web-based quiz application that helps students prepare for exams using Previous Year Questions (PYQs). The platform organizes questions by subject, topic, and difficulty level, allowing users to practice efficiently and identify their strengths and weaknesses. It provides instant feedback, score tracking, and performance analytics to help students improve their exam readiness.

The application includes features such as user authentication, timed quizzes, randomized question generation, progress tracking, and detailed result analysis. By leveraging a structured question database and a user-friendly interface, Smart PYQ Quiz offers a personalized and engaging learning experience, making exam preparation more effective and accessible.

Key Features:

-> Practice quizzes based on previous year questions.
-> Subject-wise and topic-wise question categorization.
-> Timed quiz mode with automatic scoring.
-> Instant feedback with correct answers and explanations.
-> Performance dashboard to track progress and identify weak areas.
-> User authentication and personalized learning history.
-> Responsive design for desktop and mobile devices.

Technologies Used (example):

Frontend: HTML, CSS, JavaScript, React.js
Backend: Node.js, Express.js
Database: MongoDB / MySQL
Authentication: JWT / Firebase Authentication
Version Control: Git & GitHub
